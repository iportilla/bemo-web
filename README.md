# BEMO Next.js

BEMO is a programmed chatbot who uses music to turn your emotions into words.

## Tech Stack
- **Framework:** Next.js 15
- **Styling:** Vanilla CSS (App Router)
- **Integration:** Watson Assistant Chat
- **Fonts:** Google Fonts (Outfit)

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Conversion Details
This project was converted from a static HTML/CSS site to Next.js to provide a more modern developer experience and a premium user interface. The original static files are preserved in the `legacy/` directory.
